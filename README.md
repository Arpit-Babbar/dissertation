PhD Dissertation. 

In order for this to work, you need to move "style_common.ts" to your Texmacs package directory. For Mac using the normal Texmacs , it is
/Applications/TeXmacs.app/Contents/Resources/share/TeXmacs/packages 

For Mac using Mogon, it is
/Applications/MoganResearch.app/Contents/Resources/share/Xmacs/packages

For Windows and Linux, according to https://gitlab.act.reading.ac.uk/j.wouters/tm-styles, it is

C:\Users\....\AppData\Roaming\TeXmacs\packages

and

~\.TeXmacs\packages

respectively
